Feedstock for [afterburner](https://github.com/eic/afterburner) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Monte Carlo afterburner for beam crossing-angle and beam effects

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `0.2.1` |
| linux-arm64 (`linux-aarch64`) | ✅ `0.2.1` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge afterburner
```
